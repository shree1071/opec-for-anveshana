THE PROBLEM: Career Confusion Epidemic in India

STATISTICAL EVIDENCE:

1. 72% of Indian college students are undecided about career path
   Source: NASSCOM Youth Employment Report 2023
   Impact: Wrong course selection, wasted years

2. 68% choose careers based on parental pressure, not personal interest  
   Source: FICCI Skills Gap Report
   Impact: Low job satisfaction, high attrition rates

3. 33% of engineering graduates work in unrelated fields
   Source: AICTE Employment Survey 2023
   Impact: Rs 2.5 Lakh average wasted per wrong admission

4. 85% of Tier-2/3 city students have NO access to career counselors
   Source: UNESCO Education Report
   Impact: Massive inequality of opportunity

- 10th Grade Student: "Should I take Science or Commerce? Parents say engineering..."

AGENT 1: OBSERVER AGENT (Pattern Detection)
------------------------------------------
Purpose: Detects psychological patterns that students cannot see themselves
How It Works:
- Analyzes every message for 18 behavioral signals
- Scores each signal on a 0-10 scale
- Identifies hidden patterns: external pressure, circular thinking, 
  identity uncertainty, sunk cost fallacy
Output: JSON signal scores + detected patterns
Example: Detects "My parents want me to..." = External Pressure (8/10)

AGENT 2: COUNSELOR AGENT (Empathetic Guidance)
----------------------------------------------
Purpose: Provides context-aware, empathetic career guidance
How It Works:
- Receives student profile (education level, stream, interests, goals)
- Receives detected patterns from Observer Agent
- Generates personalized responses that address both question AND patterns
- Uses student's name, references their specific context
Output: Natural language response with embedded interventions
Example: "Hey Priya, I notice you keep mentioning what your parents want. 
         Let's explore what YOU actually enjoy..."

AGENT 3: PATHWAY AGENT (Roadmap Generation)
-------------------------------------------
Purpose: Creates actionable 5-year career roadmaps
How It Works:
- Takes student profile + conversation insights
- Generates structured JSON with year-by-year progression
- Creates visual Mermaid.js flowcharts
- Includes: skills to acquire, milestones, salary projections
Output: 5-year roadmap with visualization

Example Roadmap:
+-------+-----------------------------+--------------------------------+
| Year  | Role                        | Key Skills                     |
+-------+-----------------------------+--------------------------------+
| 1     | Learn Fundamentals          | Python, Data Structures, Git   |
| 2     | Internship at Startup       | React, APIs, Team Collaboration|
| 3     | Junior Developer            | System Design, Cloud Basics    |
| 4     | Mid-Level Engineer          | Leadership, Architecture       |
| 5     | Senior Engineer / Lead      | Mentoring, Strategic Thinking  |
+-------+-----------------------------+--------------------------------+

AGENT 4: VOICE AGENT (Conversational Interface)
-----------------------------------------------
Purpose: Enables natural, hands-free career conversations
How It Works:
- Real-time speech-to-speech conversation
- Injects dynamic context variables (name, goals, detected mood)
- Maintains conversation state across the session
- Provides audio responses with natural prosody
Output: Real-time voice conversation
